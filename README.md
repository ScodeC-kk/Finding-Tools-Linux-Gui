# Finding-Tools-Linux-Gui

A very sample tool to use finding command in gui on linux 

This program run on python2.7

### Operating System:

```
Kali linux 
```

### Setup:
```
install python2.7 tkinter xrandr xdg-open
```

### How to launch:
```
python2.7 Start.py
```
